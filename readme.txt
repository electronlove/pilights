Raspberry Pi RGB Light Server
-----------------------------

A flask web app that controls RGB lights (neopixel) connected to a raspberry pi.

Dependencies:
- [flask](http://flask.pocoo.org/)
- [bower](http://bower.io/)
- [rpi_ws281x](https://github.com/jgarff/rpi_ws281x) when running on
  Raspberry Pi
